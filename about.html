About this demo

This is a tiny static blog scaffold intended as a starting point. It uses plain HTML and CSS so it can be hosted on most static hosting providers.

How to add posts

  1. Create a new HTML file under posts/, e.g. posts/another-post.html.
  2. Add a link to the new post from index.html (or build a small script to generate the index).
  3. Commit and push to your hosting provider.
